# Microsoft Intune Lab

## Overview

This repository documents the design, deployment, and management of a simulated Microsoft Intune environment built to develop hands-on endpoint administration skills.

Rather than following isolated tutorials, this project simulates the work of an Endpoint Administrator responsible for deploying, managing, and securing Windows devices in a small business.

---

## Lab Goals

- Learn Microsoft Intune
- Learn Microsoft Entra ID
- Build real-world troubleshooting skills
- Practice documenting technical work
- Create a portfolio for future employers

---

## Lab Environment

| Component | Technology |
|-----------|------------|
| Identity | Microsoft Entra ID |
| Device Management | Microsoft Intune |
| Licensing | Microsoft 365 Business Premium |
| Operating System | Windows 11 |
| Virtualization | Oracle Virtual Box |
