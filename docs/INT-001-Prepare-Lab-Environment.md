# INT-001 – Prepare the Microsoft Intune Lab Environment

## Objective

To prepare a Microsoft Intune lab environment capable of managing Windows devices through Microsoft Entra ID and Microsoft Intune. This lab serves as the foundation for future endpoint management projects within the simulated enterprise environment.

## Business Scenario

Slothz Tech Solutions currently manages Windows computers manually, which has become time-consuming as the company has grown. The IT department wants a centralized solution to deploy settings, manage devices, improve security, and reduce the time required to configure new employee laptops.

Microsoft Intune and Microsoft Entra ID were selected to provide cloud-based device management and identity services for all company-owned Windows devices.

## Environment 

| Component | Details |
|-----------|---------|
| Company | Slothz Tech Solutions |
| Microsoft License | Microsoft 365 Business Premium |
| Identity Platform | Microsoft Entra ID |
| Device Management | Microsoft Intune |
| Virtualization | Oracle Virtualbox |
| Guest Operation System | Windows 11 Enterprise |
| Host Operating System | Windows 11 Home |
| Number of Devices | One Windows 11 Enterprise Virtual Machine |

## Outcome 

The initial Microsoft Intune lab environment was prepared with one Windows 11 Enterprise virtual machine. The device was successfully connected to Microsoft Entra ID and enrolled into Microsoft Intune through automatic enrollment.

This confirms that the lab environment is ready for future endpoint management tasks, including device groups, configuration profiles, compliance policies, and application deployment.
