# Slothz Tech Solutions

## Company Overview

Slothz Tech Solutions is a fictional managed IT services and technology consulting company created for the purpose of simulating an enterprise Microsoft Intune environment.

The company is transitioning from manually managed Windows devices to Microsoft Intune and Microsoft Entra ID to improve endpoint security, standardize device configurations, and simplify device lifecycle management.
